Data: 22/05/2026

## INTRODUÇÃO
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Raphael Augusto Paulino Leite. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especificos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon S3;
- Armazenamento seguro de dados com escalamento caso necessário;
- Utilizar o serviço Amazon S3 para armazenar as informações como registros laboratoriais, de remédios, exames clínicos, documentos regulatórios, etc;

Etapa 2: 
- AWS Lambda;
- Automação de processos com execução serverless;
- Utilizar o serviço AWS Lambda para automatizar as tarefas de processamento de resultados laboratoriais, envio de notificações e atualização automática do banco de dados;

Etapa 3: 
- Amazon Quicksight;
- Business intelligence (Análise de dados);
- Utilizar o serviço Amazon Quicksight para gerar dashboards e relatórios;

## ConclusÃ£o
A implenetação de ferramentas na empresa farmacêutica *Abstergo Industries* tem como esperado: um melhor armazenamento de dados utilizando a nuvem através do Amazon S3; Automação de processos com o AWS Lambda; e a geração de dashboards e relatórios em tempo real, facilitando a tomada de decisões através do Amazon Quicksight.
Esses aspectos aumentarão a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
